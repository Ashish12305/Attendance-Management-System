# Attendance-Management-System
Attendance management system built with java swing. Features include student management, attendance tracking, Report generation, CSV/Excel export, and an easy-to-use graphical user interface.

## Project Overview:-
The Attendance Management System is a desktop application developed using Java Swing. It helps educational institutions manage student attendance efficiently through a uesr-friendly graphical interface.

## Feature-

-Student Registration
-Attendance Marking
-Attendance Records Management
-Search Student Records
-Export Attendance Data(CSV/Excel)
-User-friendly GUI
-Attendance Report Generation.

## Technologies Used
--java
--Java Swing
--Apache POI
--Jfree Chart
--iText PDF
